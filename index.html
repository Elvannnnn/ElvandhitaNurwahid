Teknologi Cloud

Contoh Layanan Cloud computing

1. Public cloud : Penggunaan jenis cloud computing yang ditujukan murni untuk umum; 2. Private cloud : Layanan ini menyediakan pengelolaan dan pengoperasian infrastruktur hanya untuk klien tertentu yang biasanya berskala besar dan memiliki dana untuk punya cloud sendiri; 3. Community cloud : Dipakai secara bersama-sama oleh sejumlah organisasi yang punya kepentingan yang sama dan kebutuhan tingkat keamanan yang sama; 4. Hybrid cloud : Meskipun secara entitas berdiri sendiri, hybrid cloud menggunakan teknologi yang memungkinkan portabilitas data dan terhubungnya aplikasi antar-cloud;
